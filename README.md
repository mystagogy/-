# 금융상품 추천 사이트 제작
미니 프로젝트 1팀 백엔드 repository

# Environment
- Spring Boot 2.7.8
- JDK 11, JAVA 11
- Spring Data JPA
- Spring security & JWT
- Lombok
- Thymeleaf
- Build Tool: gradle
- CI/CD: Jenkins 2.361.3
- Infra(PaaS): AWS EC2
- DB: MariaDB 
- Log: Slf4j
- API Docs: Swagger3
  springdoc-openapi v1.6.14
